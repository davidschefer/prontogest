🏥 ProntoGest — Sistema de Gestão Hospitalar

Sistema web desenvolvido para clínicas e pequenos hospitais, focado em organização, agilidade e controle operacional.

🚀 Funcionalidades
📋 Cadastro de pacientes
🩺 Triagem com sinais vitais (incluindo HGT)
📑 Prontuário Eletrônico (PEP)
💊 Prescrições médicas
🛏️ Controle de leitos
📅 Agenda de consultas
🧾 Faturamento
👥 Gestão de funcionários
📊 Relatórios operacionais
🛠️ Tecnologias
Frontend: HTML, CSS, JavaScript
Backend: Node.js + Express
Autenticação: JWT
Armazenamento atual: memória/localStorage (MVP)
Estrutura API-first preparada para MySQL
⚙️ Como rodar o projeto
cd backend
npm install
node server.js

Acesse:

http://localhost:3000/Html/login.html
🎯 Objetivo

Projeto voltado para uso real em clínicas de pequeno porte, com foco em:

Simplicidade
Baixo custo
Alta produtividade operacional
📌 Status

🚧 MVP funcional — em evolução para produção
